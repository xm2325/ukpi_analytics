# Generated snapshot

This folder stores one generated run from the synthetic UKPI analytics demo.

It is checked into the repository so the project is not only visible through GitHub Actions artifacts. The data here are still synthetic. They are not Vanguard data and not real customer data.

The snapshot is regenerated by:

```bash
python ukpi_analytics_demo.py
python scripts/evaluate_segmentation.py
python scripts/build_evidence_layer.py
python scripts/run_sql_checks.py
python scripts/export_generated_snapshot.py
python scripts/build_guided_dashboard.py
```

The source of truth remains the code. This folder is a reviewable frozen output from one run, including segmentation diagnostics, uncertainty estimates, data-quality checks, and SQL results.
